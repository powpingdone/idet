# IDeT

An experimential text editor. Press ctrl & space.

IDeT is a GTK-4 based text editor. It uses keybindings to interact with various actions. 

# BUILDING

Dependencies required:

* gtkmm-4

After installing the dependencies

```
meson builddir
cd builddir
ninja 
sudo ninja install
```
